# Sosheskaz Tap

## How do I install these formulae?

`brew install sosheskaz/tap/<formula>`

Or `brew tap sosheskaz/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sosheskaz/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
